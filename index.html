print("Parity Error Checking")_

A parity bit is an additional bit that is used to check that the other bits are transmitted correctly. If using 7-bit ASCII there is an extra bit available to be used for parity. The total number of 1’s is counted in each byte including the parity bit equalled an odd or even number.

ascii_code = input("Enter a 7-bit binary number: ")

total_of_is = 0
for i in range(0, 6):
    total_of_is += int(ascii_code[i])

def isEven(num):
    if (num % 2) == 0:
        return True
    else:
        return False

if isEven(total_of_is) and ascii_code[0] == "0":
    print("Data ok!")
elif not isEven(total_of_is) and ascii_code[0] == "1":
    print("Data ok!")
else:
    print("Data Corrupt :-(")


print("Majority Voting")_ 

from statistics import mode

data_1 = input("The first transfer: ")
data_2 = input("The second transfer: ")
data_3 = input("The third transfer: ")

true_data = ""
i = 0
while i < len(mode([data_1, data_2, data_3])):
    true_data = true_data + mode([data_1[i], data_2[i], data_3[i]])
    i += 1

print(true_data)

Majority voting is a system that requires each bit to be sent 3 times. If a bit value is flipped during transmission over a noisy line, the receiving computer would use the majority rule and would assume that the two bits that have not changed are correct.



print("Checksums")_

A checksum is a mathematical algorithm applied to a packet of data. The result is also sent with the data. The algorithm is applied at the receiving end and the checksum should agree with the checksum sent.

binary_string = input("Enter x5 8-bit binary values: ")

binary_string = binary_string.replace(' ', '')

checksum = int(binary_string[:8], 2)

binary_string_1 = binary_string[8:16]
binary_string_2 = binary_string[16:24]
binary_string_3 = binary_string[24:32]
binary_string_4 = binary_string[32:40]

correct_checksum = int(binary_string_1, 2) + int(binary_string_2, 2) + int(binary_string_3, 2) + int(binary_string_4, 2)

if str(correct_checksum) == str(checksum):
    print("Correct!")
else:
    print("Data is corrupt...")
