print("2D Lists & Nested Loops")_ 

x_dimensions = int(input("How many of each timetable would you like? ")) + 1
y_dimensions = int(input("\nHow many times table would you like to include? ")) + 1

for x in range(1, x_dimensions):
	print("\n")
	for y in range(1, y_dimensions):
		if len(str(y * x)) == 2:
			end_gap = "    "
		elif len(str(y * x)) == 3:
			end_gap = "   "
		else:
			end_gap = "     "

		print(str(x * y ),  end = end_gap)

A rational number is any value that can be expressed as a ratio or fraction. These can be negative. You cannot divide by zero.

print("Irrational Numbers")_ 

from decimal import Decimal

d2 = Decimal(2)

dot100 = decimal.Context(prec=100)

print(d2.sqrt(dot100))

An irrational number is a number that cannot be represented as a ratio of integers because the decimal equivalent would go on forever without repeating. An example of this is pi or any surd.
