        return True
    else:
        return False

print("Majority Voting")_ 

from statistics import mode

data_1 = input("The first transfer: ")
data_2 = input("The second transfer: ")
data_3 = input("The third transfer: ")

true_data = ""
i = 0
while i < len(mode([data_1, data_2, data_3])):
    true_data = true_data + mode([data_1[i], data_2[i], data_3[i]])
    i += 1

print(true_data)

Majority voting is a system that requires each bit to be sent 3 times. If a bit value is flipped during transmission over a noisy line, the receiving computer would use the majority rule and would assume that the two bits that have not changed are correct.

print("Binary Addition")_

Addition in binary works on the same principles as denary addition. You add the digits in the same column from right to left. If the digits together equal a number larger than the base value, then one is carried over to the next column.

num1 = input("First binary number to add: ")
num2 = input("Second binary number to add: ")

ans = bin(int(num1,2) + int(num2,2))

print(ans.replace("0b", ""))


print("Binary Multiplication")_ 

num1 = input("First binary number to multiply: ")
num2 = input("Second binary number to multiply: ")

ans = bin(int(num1,2) * int(num2,2))

print(ans.replace("0b", ""))

You use the same principle for binary multiplication as for denary multiplication, moving a number one place to the left multiplies it by the base (10 or 2). In binary, you have to split the number you want to multiply by into steps that are powers of 2. eg. 17 x 14 = 17 x 8 + 17 x 4 + 17 x 2
