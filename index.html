print("Numbers")_

In Python, the operators for numbers are +, -, *, /. To print a number you first have to convert it to a string using str(). By importing math, you get lots of extra maths functions that aren't native to Python, such as rounding, powers and roots. You convert a string to integer by using int() and you use float() for decimals.

from math import *

print(5)
print(5 * 2)
print(5 / 2)
print(5 + 2)
print(5 - 2)

my_num = 5
print(str(my_num) + " is a number.")

my_numnum = -5
print(abs(my_numnum))

print(pow(4, 64))

print(max(4, 6))
print(min(4, 6))

print(round(3.8))
print(round(3.2))

print(floor(3.8))
print(ceil(3.8))

print(sqrt(123456789))
